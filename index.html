01
PROJECT

고객 리뷰 데이터 기반 고객 경험 개선CANDEO HOTELS KYOTO KARASUMA ROKKAKU

기여도 75%

PROBLEM

고객 리뷰와 현장 인터뷰를 통해 시설 이용 방법, 체크인 안내 시간, 부대시설 정보에 대한 불편이 반복적으로 확인되었습니다.

ACTION

고객 리뷰와 현장 인터뷰 내용을 분석하고, 예약 사이트에 등록된 정보와 실제 시설 안내지를 비교 검토했습니다. 그 결과 예약 사이트별 안내 내용과 시설 안내지의 정보가 일치하지 않아 고객 안내 시간이 길어지고 있다는 점을 확인했습니다. 이에 반복적으로 발생하는 문의 항목을 정리하여 예약 사이트와 시설 안내지의 정보를 통일화했습니다. 또한 경쟁 호텔 사례를 참고하여 텍스트 중심의 안내지를 이미지와 아이콘 중심으로 시각화함으로써, 고객이 쉽고 빠르게 정보를 이해할 수 있도록 개선했습니다.

RESULT

체크인 안내 시간이 15분에서 5분으로 단축되었고, 고객 안내 방식이 명확해지면서 반복 문의와 부정 리뷰가 감소했습니다. 그 결과 고객 만족도 개선 및 2024년 AGODA 고객 리뷰 어워드 달성에 기여했습니다.

02
PROJECT

데이터 분석 기반 문제 원인 도출CANDEO HOTELS KYOTO KARASUMA ROKKAKU

기여도 80%

PROBLEM

간사이 지역 정기회의에서 리넨 세탁비가 전월 대비 급격히 증가한 것을 확인했으나, 현장에서는 정확한 원인을 파악하지 못하고 있었습니다.

ACTION

정기회의에서 공유된 운영 데이터를 검토하고, 객실 이용 현황과 고객 요청 내역을 분석했습니다. 그 결과 유럽 가족 단위 고객 비중이 증가하면서 수건 추가 요청과 객실 전체 청소 요청이 평소보다 많이 발생했다는 점을 확인했습니다. 이후 해당 내용을 정리하여 본사에 공유하고 향후 운영 계획 수립에 참고할 수 있도록 보고했습니다.

RESULT

리넨 세탁비 증가의 원인을 고객 이용 패턴과 운영 데이터를 기반으로 분석하여 본사에 보고했습니다. 이를 통해 단순 비용 증가가 아닌 고객 구성 변화에 따른 현상임을 설명하고, 운영 현황을 보다 정확하게 판단할 수 있도록 지원했습니다.
